# MoYu-2048
 无聊，娱乐一下

**使用方法：**
1. 依赖`readchar`，`pip install readchar`
2. 在终端中运行游戏：python3 2048.py
3. 游戏开始后，使用 ←, →, ↑, ↓ 进行上下左右移动方块。
4. 使用命令 c 继续游戏，r 重新开始游戏，q 退出游戏。
5. 🎮


**试玩效果**

> 平台：macOS 14.4.1
> 
> 终端：iterm2
